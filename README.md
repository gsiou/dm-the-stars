# DM The Stars Discord App

## Setup
You need a .env
```
PUBLIC_KEY=...
APPLICATION_ID=...
BOT_TOKEN=...
HOMEPAGE=https://dm-the-stars.gsiou.com
SOURCE=https://github.com/gsiou/dm-the-stars
ISSUES=https://github.com/gsiou/dm-the-stars/issues
```

Install dependencies
```
npm install
```

To setup your commands run
```
node commands.js
```

Finally, to run
```
node app.js
```
